# Sarrera

Ongi etorri Vue-ren osagaien atalera! Atal honetan, osagaien oinarrizko kontzeptuak eta haien erabilera aztertuko ditugu.

## Zer ikasiko dugu?

- Osagaien oinarrizko kontzeptuak
- Nola sortu eta erabili osagaiak
- Osagaien arteko komunikazioa
- Osagai berrerabilgarriak sortzen

## Zergatik erabili osagaiak?

- **Berrerabilgarritasuna**: Kode bera hainbat lekutan erabil daiteke
- **Mantentze errazagoa**: Kodea modu logikoan banatuta dago
- **Probaketa errazagoa**: Osagai bakoitza isolatuan proba daiteke
- **Elkarlana**: Taldeko kideek modu independentean lan egin dezakete
