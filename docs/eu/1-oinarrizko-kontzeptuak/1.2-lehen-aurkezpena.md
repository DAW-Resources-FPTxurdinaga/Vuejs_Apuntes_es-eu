# 1.2 Zure lehen Vue aplikazioa

Adibide praktiko honetan, zereginen zerrenda sinple bat sortuko dugu Vue-ren oinarrizko kontzeptuak ulertzeko.

## Proiektuaren egitura

```
nire-lehen-aplikazioa/
├── index.html
└── app.js
```

## 1. Urratsa: Hasierako konfigurazioa

Sortu `index.html` izeneko fitxategi bat ondorengo edukiarekin:

```html
<!DOCTYPE html>
<html lang="eu">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nire Lehen Vue Aplikazioa</title>
    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
    <style>
        .osatuta {
            text-decoration: line-through;
            color: #888;
        }
    </style>
</head>
<body>
    <div id="app">
        <h1>Zereginen Zerrenda</h1>
        
        <!-- Zeregin berria gehitzeko formularioa -->
        <div class="formularioa">
            <input 
                v-model="zereginBerria" 
                @keyup.enter="gehituZeregina"
                placeholder="Idatzi zeregin bat eta sakatu Sartu"
            >
            <button @click="gehituZeregina">Gehitu</button>
        </div>
        
        <!-- Zereginen zerrenda -->
        <ul>
            <li v-for="(zeregina, index) in zereginak" :key="index" 
                :class="{'osatuta': zeregina.osatuta}"
                @click="zeregina.osatuta = !zeregina.osatuta">
                {{ zeregina.testua }}
                <button @click.stop="ezabatuZeregina(index)">Ezabatu</button>
            </li>
        </ul>
        
        <!-- Zereginen kontadorea -->
        <p v-if="zereginak.length > 0">
            Osatutako zereginak: {{ osatutakoZereginak.length }} {{ zereginak.length }}-tik
        </p>
        <p v-else>
            Ez dago zereginik. Gehitu bat hasteko!
        </p>
    </div>

    <script src="app.js"></script>
</body>
</html>
```

## 2. Urratsa: Aplikazioaren logika

Sortu `app.js` izeneko fitxategi bat ondorengo kodearekin:

```javascript
const { createApp, ref, computed } = Vue;

createApp({
  setup() {
    const zereginBerria = ref('');
    const zereginak = ref([
      { testua: 'Ikasi Vue.js', osatuta: false },
      { testua: 'Erosketak egin', osatuta: true },
      { testua: 'Medikuari deitu', osatuta: false }
    ]);

    const osatutakoZereginak = computed(() => {
      return zereginak.value.filter(z => z.osatuta);
    });

    function gehituZeregina() {
      if (zereginBerria.value.trim() === '') return;
      
      zereginak.value.push({
        testua: zereginBerria.value,
        osatuta: false
      });
      
      zereginBerria.value = ''; // Garbitu sarrera-eremua
    }

    function ezabatuZeregina(index) {
      zereginak.value.splice(index, 1);
    }

    return {
      zereginBerria,
      zereginak,
      osatutakoZereginak,
      gehituZeregina,
      ezabatuZeregina
    };
  }
}).mount('#app');
```

## Adibide honetan ikasitako kontzeptuak

1. **Vue-ren zuzentarauak**:
    - `v-model`: Datuen lotura biderkakoa egiteko
    - `v-for`: Array-etan iteratzeko
    - `v-if`/`v-else`: Baldintzapeko errenderizaziorako
    - `@click`: Klik-ekarpenak kudeatzeko

2. **Erreaktibotasuna**:
    - `ref` erabiliz aldagai erreaktiboak sortzeko
    - `computed` erabiliz kalkulatutako propietateak sortzeko

3. **Metodoak**:
    - Aplikazioaren egoera kudeatzeko funtzioak

## Ariketa praktikoak

1. Gehitu botoi bat zeregin guztiak osatuta bezala markatzeko
2. Inplementatu zeregin bat editatzeko funtzionalitatea
3. Gehitu aukera zereginak iragazteko (guztiak, osatuak, zain daudenak)
4. Gorde zereginak `localStorage`-n orria berritu ondoren mantentzeko
