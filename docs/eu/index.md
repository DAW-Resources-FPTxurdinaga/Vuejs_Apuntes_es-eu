## ğŸš€ Hasi

Vue 3-rekin hasteko, jarraitu urrats hauek:

1. Joan [Oinarrizko Kontzeptuak](./oinarrizko-kontzeptuak/sarrera.md) atalera Vue 3-ren sarrera osoa ikusteko.

2. Egiaztatu [Node.js](https://nodejs.org/) (16. bertsioa edo berriagoa) eta [pnpm](https://pnpm.io/installation) edo [yarn](https://yarnpkg.com/getting-started/install) instalatuta dituzula.

3. Sortu Vue proiektu berria:
    - **npm:**
   ```
   npm create vue@latest nire-vue-proiektua
   ```
    - **yarn:**
   ```
   yarn create vue nire-vue-proiektua
   ```
    - **pnpm:**
   ```
   pnpm create vue nire-vue-proiektua
   ```
4. Sartu proiektuaren katalogoan:
   ```
   cd nire-vue-proiektua
   ```
5. Instalatu menpekotasunak:
   ```
   npm install  # edo yarn install / pnpm install
   ```
6. Abiarazi garapen-zerbitzaria:
   ```
   npm run dev  # edo yarn dev / pnpm dev
   ```

7. Ireki zure nabigatzailea `http://localhost:5173` helbidean zure aplikazioa ikusteko.
