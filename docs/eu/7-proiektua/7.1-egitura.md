## Gako Karpeten Azalpena

### 1. `/public`
- Konpilazio direktorioan zuzenean kopiatzen diren fitxategi estatikoak
- `index.html` aplikazioaren oinarri txantiloia da

### 2. `/src/components`
- **ui/**: UI osagai orokorrak eta berrerabilgarriak
  - Adib.: `Button.vue`, `Card.vue`, `Modal.vue`
- **shared/**: Orri ezberdinek partekatzen dituzten osagaiak
  - Adib.: `Layout.vue`, `Header.vue`, `Footer.vue`

### 3. `/src/composables`
- Composition API erabiliz logika berrerabilgarria
- Adib.: `useFormValidation.js`, `usePagination.js`

### 4. `/src/stores`
- Egoera globalaren kudeaketarako Pinia dendak
- Fitxategi bakoitza domeinu logiko bati dagokio

### 5. `/src/views`
- Orri osoak ordezkatzen dituzten osagaiak
- Router-ean ibilbideetara mapatuta

## Vue Osagai baten Egitura

```vue
<template>
  <div class="componente">
    <!-- Osagaiaren edukia -->
    <h1>{{ titulo }}</h1>
    <slot></slot>
  </div>
</template>

<script setup>
// Inportazioak
import { ref, computed, onMounted } from 'vue';
import { useStore } from '../stores';

// Props
defineProps({
  titulo: {
    type: String,
    required: true
  }
});

// Egoera erreaktiboa
const contador = ref(0);
const store = useStore();

// Kalkulatuak
const contadorDoble = computed(() => contador.value * 2);

// Metodoak
function incrementar() {
  contador.value++;
}

// Bizitza-zikloaren kakoak
onMounted(() => {
  console.log('Osagaia muntatuta');
});

// <script setup>-en, hemen definitutako guztia template-an erabilgarri dago
</script>

<style scoped>
.componente {
  /* Esparru lokaleko estiloak */
}
</style>
```

## Izen-emate Konbentzioak

1. **Osagaiak**: PascalCase (adib. `UserProfile.vue`)
2. **JS/CSS Fitxategiak**: kebab-case (adib. `user-service.js`)
3. **Aldagaiak eta funtzioak**: camelCase
4. **Konstanteak**: UPPER_SNAKE_CASE
5. **Pinia dendak**: `izena.mota.js` (adib. `user.store.js`)
