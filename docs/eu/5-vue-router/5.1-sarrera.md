# 5.1 Vue Routerrera Sarrera

Vue Router Vue.js-entzako bideratze-liburutegi ofiziala da. Orrialde bakarreko aplikazioak (SPA) sortzea ahalbidetzen du, ikuspegi anitzekin eta haien arteko nabigazioa orria birkargatu gabe.

## Zer da Vue Router?

Vue Router-ek honako hau ahalbidetzen du:
- Nabigazio esperientzia leuna sortzea
- Ikuspegi-aldaketen artean aplikazioaren egoera mantentzea
- Nabigatzailearen aurrera/atzera nabigazioa erabiltzea
- Autentifikazioarekin bideak babestea
- Osagaiak modu dinamikoan kargatzea

## Kontzeptu Gakoak

- **Ibilbideak (routes)**: URLak osagaiekin mapatzen dituzte
- **Ikuspegiak (views)**: Ibilbidearen arabera bistaratzen diren osagaiak
- **Nabigazioa**: Ibilbide desberdinen arteko mugimendua
- **Parametroak**: URLaren bidez pasatzen diren datuak
- **Habiaratzea (nesting)**: Ibilbideen hierarkia

## Instalazioa

Zure proiektuan Vue Router instalatzeko:

```bash
npm install vue-router@4
# edo yarn-ekin
yarn add vue-router@4
```

## Oinarrizko Konfigurazioa

1. Sortu `router/index.js` fitxategia zure proiektuan:

```javascript
import { createRouter, createWebHistory } from 'vue-router';
import HomeView from '../views/HomeView.vue';

const routes = [
  {
    path: '/',
    name: 'home',
    component: HomeView
  },
  {
    path: '/about',
    name: 'about',
    // Osagaia alfer-kargan (lazy load)
    component: () => import('../views/AboutView.vue')
  }
];

const router = createRouter({
  history: createWebHistory(process.env.BASE_URL),
  routes
});

export default router;
```

2. Erregistratu routerra `main.js` fitxategian:

```javascript
import { createApp } from 'vue';
import App from './App.vue';
import router from './router';

const app = createApp(App);
app.use(router);
app.mount('#app');
```

## Oinarrizko Osagaiak

Vue Router-ek bi osagai nagusi eskaintzen ditu:

1. **`<router-link>`**: Ibilbideen arteko nabigaziorako
   ```vue
   <router-link to="/">Hasiera</router-link>
   <router-link :to="{ name: 'about' }">Honi buruz</router-link>
   ```

2. **`<router-view>`**: Uneko ibilbideari dagokion osagaia bistaratzen du
   ```vue
   <template>
     <div id="app">
       <nav>
         <router-link to="/">Hasiera</router-link> |
         <router-link to="/about">Honi buruz</router-link>
       </nav>
       <router-view />
     </div>
   </template>
   ```
