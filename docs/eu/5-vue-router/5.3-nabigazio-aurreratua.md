# 5.3 Vue Router-ekin Nabigazio Aurreratua

## Nabigazio Guardaak (Guards)

Nabigazio guardaek ibilbideetara sarbidea kontrolatzeko eta nabigazioa baino lehen/ondoren ekintzak egiteko aukera ematen dute.

### Guardia Globala

Nabigazio bakoitzaren aurretik exekutatzen da:

```javascript
// router/index.js
router.beforeEach((to, from, next) => {
  // to: joan nahi den ibilbidea
  // from: abiatzen den ibilbidea
  // next: nabigazioarekin jarraitzeko funtzioa

  if (to.matched.some(record => record.meta.requiresAuth)) {
    if (!isAuthenticated()) {
      next({
        path: '/login',
        query: { redirect: to.fullPath }
      });
    } else {
      next();
    }
  } else {
    next();
  }
});
```

### Ibilbide Mailako Guardia

Ibilbidearen konfigurazioan definitzen da:

```javascript
const routes = [
  {
    path: '/dashboard',
    component: Dashboard,
    meta: { requiresAuth: true },
    beforeEnter: (to, from, next) => {
      if (isAuthenticated()) {
        next();
      } else {
        next('/login');
      }
    }
  }
];
```

### Osagaian Definitutako Guardiak

Osagaiaren barruan definitzen dira:

```javascript
// Vue 3 eta Composition API-rekin ohikoena guardia globalak edo ibilbide mailakoak erabiltzea da,
// baina osagai-mailako guardiak ere existitzen dira (Options API).
// <script setup> erabiliz, antzeko jokabideak lor ditzakegu composable-en bidez
// edo bizi-zikloa eta router-a erabiliz.

import { onBeforeRouteUpdate, onBeforeRouteLeave } from 'vue-router';
import { ref } from 'vue';

const userData = ref(null);
const unsavedChanges = ref(false);

const fetchUserData = (id) => {
  // Erabiltzailearen datuak kargatzeko logika
};

// beforeRouteUpdate-ren baliokide hurbila
onBeforeRouteUpdate((to, from) => {
  // Ibilbidea aldatzen denean baina osagaia berrerabiltzen denean deitzen da
  // Adibidez, parametro dinamikoak aldatzen direnean
  userData.value = null;
  fetchUserData(to.params.id);
});

// beforeRouteLeave-ren baliokide hurbila
onBeforeRouteLeave((to, from) => {
  // Osagai hau errendertzen duen ibilbidea uztear dagoenean deitzen da
  if (unsavedChanges.value) {
    const answer = confirm('Gorde gabeko aldaketak dituzu. Ziur zaude irten nahi duzula?');
    // false itzuliz gero, nabigazioa bertan behera gelditzen da
    return answer;
  }
});
```

## Bideak Alfer-kargan (Lazy Loading)

Errendimendua hobetzeko, osagaiak behar direnean soilik kargatu:

```javascript
const routes = [
  {
    path: '/about',
    name: 'about',
    component: () => import('../views/AboutView.vue')
  },
  {
    path: '/contact',
    name: 'contact',
    // Osagaiak chunk berean taldekatzea
    component: () => import(/* webpackChunkName: "contact" */ '../views/ContactView.vue')
  }
];
```

## Scroll Portaera (Scroll Behavior)

Nabigazioan scroll portaera kontrolatu:

```javascript
const router = createRouter({
  // ...
  scrollBehavior(to, from, savedPosition) {
    // Itzuli nahi den posizioa
    if (savedPosition) {
      return savedPosition;
    } else if (to.hash) {
      return {
        el: to.hash,
        behavior: 'smooth',
      };
    } else {
      return { top: 0 };
    }
  },
});
```

## Trantsizioak

Ibilbideen arteko trantsizioei animazioak gehitu:

```vue
<template>
  <router-view v-slot="{ Component }">
    <transition name="fade" mode="out-in">
      <component :is="Component" />
    </transition>
  </router-view>
  </template>

<style>
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s ease;
}

.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
</style>
```
