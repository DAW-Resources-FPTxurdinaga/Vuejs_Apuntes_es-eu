# 2.1 Interpolazioa eta Zuzentarau Oinarrizkoak

## Testuaren Interpolazioa

Datuak lotzeko modurik errazena testu-interpolazioaren sintaxia erabiltzea da ("Mustache" izenekoa, `{{ }}` kortxeteen ondorioz):

```vue
<template>
  <p>Mezua: {{ mezua }}</p>
  <p>Emaitza: {{ zenbakia * 2 }}</p>
  <p>Adin nagusikoa al da? {{ adina >= 18 ? 'Bai' : 'Ez' }}</p>
</template>

<script setup>
const mezua = 'Kaixo Vue!';
const zenbakia = 5;
const adina = 20;
</script>
```

## v-text Zuzentaraua

`{{ }}` interpolazioaren alternatiba:

```vue
<template>
  <p v-text="'Kaixo ' + izena"></p>
</template>

<script setup>
const izena = 'Erabiltzailea';
</script>
```

## v-html Zuzentaraua

HTML-a ihesik gabe txertatzeko:

```vue
<template>
  <div v-html="htmlGordina"></div>
</template>

<script setup>
import { ref } from 'vue';

const htmlGordina = ref('<span style="color: red">Testu gorria</span>');
</script>
```

## v-bind Zuzentaraua

HTML atributuak lotzeko:

```vue
<template>
  <div :class="{ aktiboa: aktiboaDago }">
    Eduki dinamikoa
  </div>
  <img :src="irudiarenUrl" :alt="testuAlternatiboa">
</template>

<script setup>
const aktiboaDago = true;
const irudiarenUrl = '/bide-helbidea/irudia.jpg';
const testuAlternatiboa = 'Irudiaren deskribapena';
</script>
```

## Ariketa Praktikoa

Sortu profil-txartel bat erakusten duen osagaia honekin:
1. Izena eta abizena interpolatuta
2. Irudi bat v-bind erabiliz
3. HTML onartzen duen deskribapena
4. Egoera adierazlea (aktiboa/ez-aktiboa)

```vue
<template>
  <div class="profil" :class="{ 'aktiboa': erabiltzailea.aktiboa }">
    <img :src="erabiltzailea.argazkia" :alt="'Irudia: ' + erabiltzailea.izena">
    <h2>{{ erabiltzailea.izena }} {{ erabiltzailea.abizena }}</h2>
    <div v-html="erabiltzailea.deskribapena"></div>
    <p>Egoera: {{ erabiltzailea.aktiboa ? 'Aktiboa' : 'Ez-aktiboa' }}</p>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const erabiltzailea = ref({
  izena: 'Ane',
  abizena: 'Martinez',
  argazkia: 'https://via.placeholder.com/150',
  deskribapena: '<strong>Frontend</strong> garatzailea Vue esperientziarekin',
  aktiboa: true
});
</script>

<style scoped>
.profil {
  border: 1px solid #ddd;
  padding: 1rem;
  border-radius: 8px;
  max-width: 300px;
}

.profil.aktiboa {
  border-color: #42b983;
  background-color: #f8f9fa;
}

img {
  max-width: 100%;
  border-radius: 4px;
  margin-bottom: 1rem;
}
</style>
```
