# Egoeraren Kudeaketa Pinia-rekin

## 6.1 Pinia-ri Sarrera

Pinia Vue-ren egoera kudeatzeko irtenbide ofiziala da. Egoeraren kudeaketarako liburutegia da, Vuex-ek (aurreko sistema) baino API sinpleagoa eta tipatua eskaintzen duena.

### Zergatik erabili Pinia?

- **Tipatze hobea**: TypeScript-entzako lehen mailako euskarria
- **API sinpleagoa**: Kode errepikapena gutxiago
- **Moduluak lehenetsita**: Ez du habiaratutako modulurik behar
- **DevTools**: Bateragarritasuna Vue DevTools-ekin
- **Composable API**: Composition API-rekin lan egiteko diseinatua
- **Arina**: Tamaina oso txikia

### Instalazioa

Vue 3 proiektuan Pinia instalatzeko, exekutatu:

```bash
npm install pinia
# edo yarn-ekin
# yarn add pinia
# edo pnpm-ekin
# pnpm add pinia
```

### Oinarrizko Konfigurazioa

Zure fitxategi nagusian (oro har, `main.js` edo `main.ts`):

```javascript
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'

const app = createApp(App)

app.use(createPinia())
app.mount('#app')
```

Prest zaude Pinia zure Vue 3 aplikazioan erabiltzen hasteko! Hurrengo ataletan, dendak (stores), actions, getters eta gehiago sortu eta erabiltzen ikusiko dugu.

## Hurrengo urratsa

Pinia-ri sarrera eginda, hurrengo pausoa da [Pinia-ko Dendak](./6.2-dendak.md) ezagutzea; bertan, dendak nola sortu eta erabili ikasiko duzu aplikazioaren egoera kudeatzeko.
