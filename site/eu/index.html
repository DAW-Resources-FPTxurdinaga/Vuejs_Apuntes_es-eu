    üöÄ Vue.js 3 Ikastaroa¶

    Ongi etorri Vue 3 ikastarora - JavaScript-eko framework progresiboa, erabiltzaile-interfaze moderno eta erreaktiboak sortzeko.

    üìö Ikastaroaren Egitura¶

    1. Oinarrizko Kontzeptuak - Vue 3-ren oinarrizko kontzeptuak
    2. Oinarriak - Framework-aren ezaugarri nagusiak
    3. Osagaiak - Osagaiak sortu eta komunikatu
    4. HTTP Eskariak - API-ak erabiltzea
    5. Vue Router - Orrialde bakarreko aplikazioak
    6. Pinia - Egoeraren kudeaketa
    7. Proiektu Praktikoa - Aplikazio osoa

    üéØ Ikasgai Helburuak¶

    • Vue 3-ren oinarrizko kontzeptuak menderatzea
    • Berreskura daitezkeen osagaiak garatzea
    • Egoeraren kudeaketa inplementatzea Pinia erabiliz
    • Orri bakarreko aplikazioak sortzea Vue Router-ekin
    • REST API-ak modu eraginkorrean erabiltzea
    • Industrian erabiltzen diren diseinu-patroiak aplikatzea

    üõ†Ô∏è Aurreko Ezagutzak¶

    • HTML5 eta CSS3 ezagutza sendoa
    • JavaScript (ES6+) esperientzia
    • Objektuei orientatutako programazioaren oinarriak
    • Garapen-tresnen ezagutza (Node.js, npm/yarn)

    üìñ Eduki Nagusiak¶

    • Vue 3 Composition API - Osagaiak idazteko modu modernoa
    • Erreaktibotasuna - Sistema eraginkorra eta indartsua
    • Fitxategi bakarreko osagaiak (.vue)
    • Garapen-tresnak - Vue DevTools, Vite, etab.

    üöÄ Hasi¶

    Vue 3-rekin hasteko, jarraitu urrats hauek:

    1. Joan Oinarrizko Kontzeptuak atalera Vue 3-ren sarrera osoa ikusteko.

    2. Egiaztatu Node.js (16. bertsioa edo berriagoa) eta pnpm edo yarn instalatuta dituzula.

    3. Sortu Vue proiektu berria:

      • npm: npm create vue@latest nire-vue-proiektua
      • yarn: yarn create vue nire-vue-proiektua
      • pnpm: pnpm create vue nire-vue-proiektua
    4. Sartu proiektuaren katalogoan: cd nire-vue-proiektua
    5. Instalatu menpekotasunak: npm install # edo yarn install / pnpm install

    6. Abiarazi garapen-zerbitzaria: npm run dev # edo yarn dev / pnpm dev

    7. Ireki zure nabigatzailea http://localhost:5173 helbidean zure aplikazioa ikusteko.
