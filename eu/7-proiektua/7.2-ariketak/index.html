    7.2 Ariketa Praktikoak¶

    Ariketa 1: Zereginen Zerrenda Hobetsia¶

    Helburua: Zereginen zerrenda aplikazio bat sortzea honako ezaugarriekin:

    1. Zeregin berriak gehitu
    2. Zereginak osatuta bezala markatu
    3. Iragazkiak (guztiak, aktiboak, osatuak)
    4. Dauden zereginak editatu
    5. Zereginak ezabatu
    6. localStorage-n persistentzia

    Baldintza Teknikoak: - Composition API erabili - Pinia denda bat inplementatu - Berrerabilgarriak diren osagaiak erabili - Diseinu erantzunkorra

    Gomendatutako Pausoak: 1. Vite + Vue 3 proiektu berria konfiguratu 2. Beharrezko osagaiak sortu 3. Dendaren logika inplementatu 4. Beharrezko estiloak gehitu

    Ariketa 2: Erabiltzaileen Bilaketa APIrekin¶

    Helburua: GitHub-eko erabiltzaileak bilatu eta haien profilak ikusteko aplikazioa sortzea.

    Ezaugarriak: - Erabiltzaile-izenaren araberako bilaketa eremua - Emaitzen zerrenda avatar eta erabiltzaile-izenarekin - Erabiltzailean klik egiterakoan xehetasun ikuspegia - Karga egoerak eta erroreen kudeaketa

    Erabili beharreko APIa:

    GET https://api.github.com/search/users?q={query}
GET https://api.github.com/users/{username}

    Bonus: - Emaitzen orrikatzea - Bilaketen cache-a - Azken bilaketen historiala

    Ariketa 3: Online Denda¶

    Helburua: Produktu katalogo bat garatu, erosketarako saskiarekin.

    Baldintzak: 1. Produktuen zerrenda irudiekin 2. Kategoria bidezko iragazkia 3. Produktuen bilaketa 4. Erosketa saski funtzionala 5. Checkout formularioa (simulatua)

    Datu Egitura Adibidea:

    const produktuak = [
  {
    id: 1,
    izena: 'Gaming Ordenagailua',
    prezioa: 1200,
    kategoria: 'teknologia',
    irudia: 'laptop.jpg',
    deskribapena: 'Jokoetarako ordenagailu indartsua'
  },
  // beste produktu batzuk...
];

    Ariketa 4: Administrazio Panela¶

    Ondoko atalak dituen administrazio panela sortu:

    1. Aginte Panela (Dashboard):

      • Estatistika orokorrak
      • Chart.js bidezko grafikoak
      • Azken jarduera

    2. Erabiltzaileen Kudeaketa:

      • Erabiltzaileen zerrenda
      • Erabiltzaileak sortu/editatu/ezabatu
      • Iragazkiak eta bilaketa

    3. Edukien Kudeaketa:

      • Artikuluen CRUD
      • Irudien igoera
      • Testu aberastuko editorea (Tiptap edo antzekoa)

    Gomendatutako Teknologiak

    • Nabigaziorako Vue Router
    • Egoera globalerako Pinia
    • HTTP eskaeretarako Axios
    • UI osagaietarako Vuetify edo PrimeVue

    Ariketa 5: Memoria Jokoa¶

    Honako ezaugarriak dituen memoria jokoa inplementatu:

    1. Kartazko taula (adib.: 4x4, 6x6)
    2. Txandaka kartak biratzea
    3. Bikote berdinak aurkitzea
    4. Mugimendu kopurua
    5. Tenporizadorea
    6. Puntuazio onenak (localStorage-n gordeta)

    Gehigarriak: - Zailtasun maila ezberdinak - Soinu efektuak - Kartak biratzerakoan animazioak - Jokalarien arteko modua

    Azken Proiektua: Oharren Aplikazioa¶

    Ezaugarri hauek dituen oharren aplikazio osoa garatu:

    Backend (JSON Server edo antzekoa erabilita simulaturik):¶

    • Erabiltzaileen autentifikazioa
    • Oharren CRUD
    • Kategoriak/etiketak
    • Bilaketa eta iragazkia

    Frontend:¶

    1. Autentifikazioa:
    2. Erregistroa eta saio-hasiera
    3. Pasahitza berreskuratzea

    4. Erabiltzaile profila

    5. Oharrak:

    6. Ohar zerrenda aurrebistarekin
    7. Ohar editore aberastua
    8. Antolaketa karpeten/etiketen arabera
    9. Bilaketa eta iragazkiak

    10. Oharrak esportatu/inportatu

    11. Ezarpenak:

    12. Gaia (argia/iluna)
    13. Tipografia
    14. Jakinarazpenak

    Erabiliko diren Teknologiak:¶

    • Vue 3 + Composition API
    • Vue Router
    • Egoera kudeaketarako Pinia
    • UI-rako Vuetify/Quasar
    • HTTP eskaeretarako Axios
    • Oharretarako Markdown
    • Offline biltegirako IndexedDB

    Proiektu bakoitza honako irizpideen arabera ebaluatuko da:

    1. Funtzionaltasuna (40%):
    2. Baldintzen betetzea
    3. Akatsen zuzentasuna

    4. Errendimendua

    5. Kodea (30%):

    6. Egitura eta antolaketa
    7. Irakurgarritasuna
    8. Praktika onak

    9. Erroreen kudeaketa

    10. UI/UX (20%):

    11. Diseinu erakargarria
    12. Erabilgarritasuna

    13. Diseinu erantzunkorra

    14. Dokumentazioa (10%):

    15. README.md
    16. Kodeko iruzkinak

    Baliabide Gehigarriak¶

    1. Vue 3 Dokumentazioa
    2. Pinia Dokumentazioa
    3. Vue Router
    4. VueUse - Vue 3-rako utilitate bilduma
    5. Vite - Eraikuntza tresna

    Entregagaiak¶

    Ariketa bakoitzerako entregatu: 1. Iturburu kodea Git biltegian 2. Instalazio eta exekuzio argibideak 3. Pantaila-argazkiak edo demo bizia (aukerakoa)

    Maiz Egindako Galderak¶

    Nola kudeatu nire aplikazioaren egoera globala?¶

    Erabili Pinia egoera kudeaketarako. Sortu denda logikoak (adib. useAuthStore, useNotesStore) eta erabili osagaietan.

    Zein da HTTP eskaerak egiteko modu onena?¶

    Erabili Axios edo Fetch API. Sortu zerbitzu edo composable bat API deialdi guztiak kudeatzeko.

    Nola inplementatu autentifikazioa?¶

    1. Sortu autentifikazio denda bat Pinia-rekin
    2. Erabili JWT saioak kudeatzeko
    3. Babestu ibilbideak router-eko guardekin
    4. Gorde token-a cookie seguruetan

    Nola hobetu aplikazioaren errendimendua?¶

    • Erabili v-once eduki estatikoarentzat
    • Inplementatu lazy loading ibilbideetan
    • Erabili v-memo zerrenda handietarako
    • Baliatu konposizioa logika berrerabilgarria izateko
